Formula Checker

Connectives

  • ¬ negation — type ~ or -
  • ∧ conjunction — type & or /\
  • ∨ disjunction — type | or \/
  • → conditional — type -> or >
  • ↔ biconditional (abbreviation) — type <-> or <>

Sentence Letters

Single letters p–z, optionally with numeric subscripts: p1, q2.

Truth Table Conventions

Columns: sentence letters (left) | subformulas | full formula (highlighted). Rows: rightmost letter alternates fastest (T, F, T, F …).

Check Mode

Formula — classifies a single formula as tautology, contradiction, or contingent.

Consistency — checks whether a set of formulas has at least one satisfying assignment.

Validity — checks whether an argument (premises + conclusion) is valid (no counterexample).

Build Mode

Enter a formula and fill in the truth table cell by cell. Click a cell to cycle through T, F, or blank. Hit Check to see which cells are correct.
