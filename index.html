How to use

  • Formulas card — enter one or more propositional formulas. Use Add formula for multiple. Examples show common patterns.
  • Build card — type a single formula and construct its truth table by clicking cells. Sentence-letter columns are pre-filled. Click each ? to cycle T → F → blank. Hit Check to verify your answers.
  • Check card — checks all formulas from the Formulas card for consistency. Consistent: highlights every row where all formulas are simultaneously true. Inconsistent: no such row exists.

Column order

Sentence letters (leftmost alternates slowest) · subformulas (depth-first) · main connective (highlighted in amber)
